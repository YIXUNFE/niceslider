# NiceSlider 1.0.0

### 特征：

- 依赖jQuery/Zepto。
- 自带控制按钮与索引值；
- 自定义索引值；
- 支持多slider嵌套；
- 双向滑动锁定；
- 单向滑动锁定；
- 自定义偏移值；
- 支持无缝循环；
- 支持重置刷新；
- 自定义动画；
- 支持回弹；
- 自动播放；
- 手势拖拽；
- 链式调用。
